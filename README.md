# 🤖 بوت مجتمع A7MED — النسخة 3.0

البوت الرسمي لسيرفر **مجتمع A7MED** على ديسكورد.

**👑 المالك:** A7MED | **🛠️ المطور:** 9ATTOS

---

## 🚀 طريقة التشغيل المحلي

```bash
# 1. تثبيت المكتبات
npm install

# 2. نسخ ملف المتغيرات وتعبئته
cp .env.example .env
# افتح .env وأضف قيمك

# 3. تشغيل البوت
npm start
```

## ☁️ النشر على Railway

1. **ادفع الكود** إلى GitHub
2. **أنشئ مشروعاً جديداً** على [Railway](https://railway.app) واربطه بالـ repo
3. **أضف المتغيرات** في لوحة التحكم: `Settings → Variables`
4. Railway سيشغّل البوت تلقائياً بـ `npm start`

### 🔑 المتغيرات المطلوبة على Railway

| المتغير | الشرح | مثال |
|---------|-------|-------|
| `DISCORD_TOKEN` | ⚠️ **مطلوب** — توكن البوت من Developer Portal | `MTI3...` |
| `DISCORD_CLIENT_ID` | معرّف تطبيق البوت | `123456789` |
| `GUILD_ID` | معرّف السيرفر الرئيسي | `987654321` |
| `DEV_ID` | معرّف حساب 9ATTOS (يُستخدم بدلاً من OWNER_ID) | `444555666` |
| `AUTO_ROLE_ID` | معرّف الرتبة التلقائية عند الانضمام | `112233445` |
| `WELCOME_CHANNEL_ID` | معرّف قناة الترحيب | `556677889` |
| `TICKET_CATEGORY_ID` | فئة قنوات التذاكر (اختياري) | `998877665` |
| `TICKET_LOG_CHANNEL_ID` | قناة سجل التذاكر (اختياري) | `554433221` |
| `PREFIX` | بادئة الأوامر (افتراضي: `!`) | `!` |
| `STARTING_BALANCE` | رصيد الأعضاء الجدد (افتراضي: `1000`) | `1000` |
| `DAILY_REWARD` | المكافأة اليومية (افتراضي: `500`) | `500` |
| `QUIZ_WRONG_PENALTY` | خصم الإجابة الخاطئة (افتراضي: `50`) | `50` |

> 💡 **ملاحظة:** `OWNER_ID` و `ADMIN_ROLE_ID` تمت إزالتهما. البوت يكتشف الأدمن تلقائياً عبر صلاحية **Administrator** في ديسكورد.

> 💡 انظر `.env.example` للقائمة الكاملة مع الشرح.

## 📚 الأوامر الكاملة

### 🛡️ الإشراف
| الأمر | الشرح |
|-------|-------|
| `!ban @عضو [سبب]` | حظر عضو |
| `!kick @عضو [سبب]` | طرد عضو |
| `!mute @عضو [مدة] [سبب]` | كتم عضو (10m, 1h, 1d) |
| `!unmute @عضو` | فك الكتم |
| `!warn @عضو [سبب]` | تحذير عضو |
| `!warns @عضو` | عرض التحذيرات |

## ✨ الميزات التلقائية
- ✅ رتبة تلقائية عند الانضمام
- ✅ رسالة ترحيب عربية مع 1000 عملة هدية + اسم من دعاك
- ✅ نظام XP تلقائي بالدردشة + مكافأة ترقية المستوى
- ✅ نقاط تلقائية: 1 نقطة/رسالة · 25 نقطة/دعوة · 5 نقاط/تذكرة
- ✅ تتبع الدعوات تلقائياً عند انضمام أعضاء جدد
- ✅ نظام كولداون لمنع السبام
- ✅ جميع الأوامر بالعربي ولها اختصارات عربية
- ✅ جميع التفاعلات تستخدم أزرار ديسكورد
- ✅ اكتشاف تلقائي للأدمن عبر صلاحيات ديسكورد (لا حاجة لـ ADMIN_ROLE_ID)

## 🎫 إعداد نظام التذاكر

1. أنشئ فئة (Category) للتذاكر في سيرفرك
2. أضف معرّف الفئة في `TICKET_CATEGORY_ID` (اختياري)
3. أضف معرّف قناة السجل في `TICKET_LOG_CHANNEL_ID` (اختياري)
4. في القناة التي تريد وضع زر التذاكر فيها، اكتب: `!ticketsetup`
5. سيظهر embed مع زر **📩 إنشاء تذكرة** للأعضاء
