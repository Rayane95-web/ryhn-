# 🤖 بوت مجتمع A7MED — النسخة 2.0

البوت الرسمي لسيرفر **مجتمع A7MED** على ديسكورد.

**👑 المالك:** A7MED | **🛠️ المطور:** 9ATTOS

---

## 🚀 طريقة التشغيل

```bash
npm install
npm start
```

## ⚙️ الإعداد (config.js)

| الحقل | الشرح |
|-------|-------|
| `token` | توكن البوت |
| `ownerId` | ID حساب A7MED |
| `devId` | ID حساب 9ATTOS |
| `adminRoleId` | ID رتبة الأدمن |
| `welcomeChannelId` | ID قناة الترحيب |
| `autoRoleId` | ID الرتبة التلقائية |

## 📚 الأوامر الكاملة

### 🛡️ الإشراف
| الأمر | الشرح |
|-------|-------|
| `!ban @عضو [سبب]` | حظر عضو |
| `!kick @عضو [سبب]` | طرد عضو |
| `!mute @عضو [مدة] [سبب]` | كتم عضو (10m, 1h, 1d) |
| `!unmute @عضو` | فك الكتم |
| `!warn @عضو [سبب]` | تحذير عضو |
| `!warns @عضو` | عرض التحذيرات |
| `!clear [عدد]` | حذف الرسائل |

### 🎭 الرتب
| الأمر | الشرح |
|-------|-------|
| `!giverole @عضو @رتبة` | منح رتبة |
| `!removerole @عضو @رتبة` | سحب رتبة |
| `!removeallroles @عضو` | إزالة كل الرتب |
| `!listroles @عضو` | عرض رتب العضو |

### 💰 الاقتصاد
| الأمر | الشرح |
|-------|-------|
| `!balance [@عضو]` | عرض الرصيد |
| `!daily` | مكافأة يومية (500 عملة) |
| `!pay @عضو [مبلغ]` | تحويل عملات |
| `!richest` | قائمة أغنى الأعضاء |

### 🎮 الألعاب
| الأمر | الشرح |
|-------|-------|
| `!qst` | سؤال عشوائي (20+ سؤال) |
| `!marathon` | ماراثون 5 أسئلة |
| `!rps [اختيار] [رهان]` | حجر ورقة مقص |
| `!8ball [سؤال]` | الكرة السحرية |
| `!coinflip` | رمي عملة |
| `!meme` | ميم عشوائي |

### ⚙️ لوحة الأدمن
| الأمر | الشرح |
|-------|-------|
| `!admin` | عرض لوحة التحكم |
| `!admin addmoney @عضو [مبلغ]` | إضافة عملات |
| `!admin removemoney @عضو [مبلغ]` | خصم عملات |
| `!admin setmoney @عضو [مبلغ]` | تعيين الرصيد |
| `!admin resetmoney @عضو` | إعادة تعيين الرصيد |
| `!admin announce #قناة [رسالة]` | إرسال إعلان |
| `!admin dm @عضو [رسالة]` | رسالة خاصة |
| `!admin clearwarns @عضو` | مسح التحذيرات |
| `!admin botinfo` | معلومات البوت |
