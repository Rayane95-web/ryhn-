# 🤖 بوت مجتمع A7MED — النسخة 2.0

البوت الرسمي لسيرفر **مجتمع A7MED** على ديسكورد.

**👑 المالك:** A7MED | **🛠️ المطور:** 9ATTOS

---

## 🚀 طريقة التشغيل المحلي

```bash
# 1. تثبيت المكتبات
npm install

# 2. نسخ ملف المتغيرات وتعبئته
cp .env.example .env
# افتح .env وأضف قيمك

# 3. تشغيل البوت
npm start
```

## ☁️ النشر على Railway

1. **ادفع الكود** إلى GitHub
2. **أنشئ مشروعاً جديداً** على [Railway](https://railway.app) واربطه بالـ repo
3. **أضف المتغيرات** في لوحة التحكم: `Settings → Variables`
4. Railway سيشغّل البوت تلقائياً بـ `npm start`

### 🔑 المتغيرات المطلوبة على Railway

| المتغير | الشرح | مثال |
|---------|-------|-------|
| `DISCORD_TOKEN` | ⚠️ **مطلوب** — توكن البوت من Developer Portal | `MTI3...` |
| `DISCORD_CLIENT_ID` | معرّف تطبيق البوت | `123456789` |
| `GUILD_ID` | معرّف السيرفر الرئيسي | `987654321` |
| `OWNER_ID` | معرّف حساب A7MED | `111222333` |
| `DEV_ID` | معرّف حساب 9ATTOS | `444555666` |
| `ADMIN_ROLE_ID` | معرّف رتبة الأدمن | `777888999` |
| `AUTO_ROLE_ID` | معرّف الرتبة التلقائية عند الانضمام | `112233445` |
| `WELCOME_CHANNEL_ID` | معرّف قناة الترحيب | `556677889` |
| `PREFIX` | بادئة الأوامر (افتراضي: `!`) | `!` |
| `STARTING_BALANCE` | رصيد الأعضاء الجدد (افتراضي: `1000`) | `1000` |
| `DAILY_REWARD` | المكافأة اليومية (افتراضي: `500`) | `500` |
| `QUIZ_WRONG_PENALTY` | خصم الإجابة الخاطئة (افتراضي: `50`) | `50` |

> 💡 انظر `.env.example` للقائمة الكاملة مع الشرح.

## 📚 الأوامر الكاملة

### 🛡️ الإشراف
| الأمر | الشرح |
|-------|-------|
| `!ban @عضو [سبب]` | حظر عضو |
| `!kick @عضو [سبب]` | طرد عضو |
| `!mute @عضو [مدة] [سبب]` | كتم عضو (10m, 1h, 1d) |
| `!unmute @عضو` | فك الكتم |
| `!warn @عضو [سبب]` | تحذير عضو |
| `!warns @عضو` | عرض التحذيرات |
| `!clear [عدد]` | حذف الرسائل |

### 🎭 الرتب
| الأمر | الشرح |
|-------|-------|
| `!giverole @عضو @رتبة` | منح رتبة |
| `!removerole @عضو @رتبة` | سحب رتبة |
| `!removeallroles @عضو` | إزالة كل الرتب |
| `!listroles @عضو` | عرض رتب العضو |

### 💰 الاقتصاد
| الأمر | الشرح |
|-------|-------|
| `!balance [@عضو]` | عرض الرصيد |
| `!daily` | مكافأة يومية (500 عملة) |
| `!pay @عضو [مبلغ]` | تحويل عملات |
| `!richest` | قائمة أغنى الأعضاء |

### 🎮 الألعاب
| الأمر | الشرح |
|-------|-------|
| `!qst` | سؤال عشوائي (20+ سؤال) |
| `!marathon` | ماراثون 5 أسئلة |
| `!rps [اختيار] [رهان]` | حجر ورقة مقص |
| `!8ball [سؤال]` | الكرة السحرية |
| `!coinflip` | رمي عملة |
| `!meme` | ميم عشوائي |

### ⚙️ لوحة الأدمن
| الأمر | الشرح |
|-------|-------|

| `!admin addmoney @عضو [مبلغ]` | إضافة عملات |
| `!admin removemoney @عضو [مبلغ]` | خصم عملات |

| `!admin resetmoney @عضو` | إعادة تعيين الرصيد |
| `!admin announce #قناة [رسالة]` | إرسال إعلان |
| `!admin dm @عضو [رسالة]` | رسالة خاصة |
| `!admin clearwarns @عضو` | مسح التحذيرات |
| `!admin botinfo` | معلومات البوت |
